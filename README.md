# 💰 Personal Finance & Expense Analytics App

A modern, production-grade **React application** to track income, expenses, budgets, and gain meaningful financial insights through interactive analytics.

---

## 🚀 Features

* 📊 Track **income & expenses** with ease
* 🧾 Add, edit, and delete transactions
* 🔍 Search and filter by category, title, or notes
* 📅 Sort transactions by date, amount, or category
* 💰 Monthly **budget tracking with progress bar**
* 🌍 View financial data in **multiple currencies**
* 💱 Built-in **currency converter** (real-time rates via API)
* 📈 Visual analytics:

  * Pie chart (category-wise spending)
  * Line chart (monthly trends)
  * Bar chart (income vs expenses)
* 🔁 Recurring expense tracking
* 💾 Data persistence using **localStorage**
* 📱 Fully responsive (mobile-friendly UI)

---

## 🧱 Tech Stack

* ⚡ **Vite + React 18**
* 🎨 **Tailwind CSS**
* 🔀 **React Router DOM**
* 🧠 **React Hook Form + Yup**
* 📊 **Recharts**
* 🎞️ **Framer Motion**
* 📅 **date-fns**
* 🔑 **uuid**
* 🔔 **react-toastify**
* 🎯 **react-icons**
* 🌐 **axios** (currency exchange API)
